# Довідник приймальної комісії ДДАЕУ — Вступна кампанія 2026

Веб-застосунок (одна сторінка) з матеріалами для приймальної комісії: склад комісій, ключові терміни, конкурсний бал, вартість навчання, нормативна база, роль керівника, дорожня карта, ЄДЕБО, профорієнтація, результати набору-2025, глосарій.

## Що в пакеті
- **index.html** — увесь застосунок. Стилі, скрипти та логотип вбудовані у файл.
- **og-image.png** — картинка прев'ю (1200×630) для месенджерів і соцмереж.
- **README.md** — ця інструкція.

> Шрифти підвантажуються з **Google Fonts** (Montserrat, Fira Sans) — потрібен інтернет. Без нього застосунок працює системними шрифтами. Для повністю автономного варіанта шрифти можна вбудувати у файл (за запитом).

## Як задеплоїти

### Варіант 1. Статичний хостинг (найпростіше)
Завантажте **обидва файли** (`index.html` і `og-image.png`) в один каталог:
- **GitHub Pages** — покладіть файли в репозиторій -> Settings -> Pages -> Branch: main -> Save.
- **Netlify / Vercel / Cloudflare Pages** — перетягніть файли (або весь архів) у панель завантаження.

### Варіант 2. Веб-сервер університету
Скопіюйте `index.html` і `og-image.png` в один каталог сайту (наприклад, `/pk2026/`).
Сторінка відкриється за адресою типу `https://www.dsau.dp.ua/pk2026/`.

### Варіант 3. Локально / без сервера
Відкрийте `index.html` подвійним кліком у браузері.

## Прев'ю при пересиланні посилання в месенджері

Коли ви ділитесь **посиланням** (після деплою), месенджери (Telegram, Viber, Facebook, Signal тощо) показують картку: велику назву, дрібніший опис і логотип. Це вже налаштовано — у файлі є мета-теги Open Graph і картинка `og-image.png`.

**Щоб прев'ю гарантовано працювало в усіх застосунках, після деплою зробіть один крок:** пропишіть **повні (абсолютні) URL** замість відносних. Відкрийте `index.html` і замініть:

    <meta property="og:image" content="og-image.png">
    <meta name="twitter:image" content="og-image.png">

на ваш домен, наприклад:

    <meta property="og:image" content="https://www.dsau.dp.ua/pk2026/og-image.png">
    <meta name="twitter:image" content="https://www.dsau.dp.ua/pk2026/og-image.png">

Бажано додати й адресу сторінки:

    <meta property="og:url" content="https://www.dsau.dp.ua/pk2026/">

Поради:
- Деякі месенджери кешують прев'ю. Якщо стара картка "залипла", скористайтесь валідатором Facebook Sharing Debugger (developers.facebook.com/tools/debug) — кнопка Scrape Again.
- Якщо переслати **сам файл** `.html` (а не посилання) — буде звичайне вкладення без картки. Прев'ю працює лише для посилань.

## Експорт у PDF / друк
У застосунку, угорі праворуч — кнопка **"Друк / PDF"**: розгортає всі розділи в один документ, далі "Зберегти як PDF".
