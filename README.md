# Довідник приймальної комісії ДДАЕУ — Вступна кампанія 2026

Веб-застосунок (одна сторінка) з матеріалами для приймальної комісії: склад комісій, ключові терміни, конкурсний бал, вартість навчання, нормативна база, роль керівника, дорожня карта, ЄДЕБО, профорієнтація, результати набору-2025, глосарій.

## Що в пакеті
- **index.html** — увесь застосунок (стилі, скрипти й логотип вбудовані).
- **og-image.png** — картинка прев'ю (1200×630) для месенджерів і соцмереж.
- **apple-touch-icon.png** — іконка для додавання на початковий екран iPhone/iPad.
- **icon-192.png**, **icon-512.png** — іконки для Android / встановлення як застосунку.
- **manifest.json** — web-маніфест (назва, іконки, повноекранний режим).
- **README.md** — ця інструкція.

> Усі файли мають лежати в **одному каталозі**. Шрифти підвантажуються з Google Fonts (потрібен інтернет; без нього — системні шрифти).

## Як задеплоїти

### Статичний хостинг (найпростіше)
Завантажте **всі файли пакета** в один каталог:
- **GitHub Pages** — у репозиторій -> Settings -> Pages -> Branch: main -> Save.
- **Netlify / Vercel / Cloudflare Pages** — перетягніть файли (або архів) у панель.

### Веб-сервер університету
Скопіюйте всі файли в один каталог сайту (наприклад, `/pk2026/`).
Відкриється за адресою `https://www.dsau.dp.ua/pk2026/`.

## Іконка-додаток на iPhone (iPhone 12 Pro Max, iOS)

Після деплою (застосунок має відкриватися за **https://**-посиланням):

1. Відкрийте посилання в **Safari** (саме Safari, не Chrome).
2. Натисніть кнопку **«Поділитися»** (квадрат зі стрілкою вгору, унизу екрана).
3. Прокрутіть і виберіть **«На початковий екран» / "Add to Home Screen"**.
4. За потреби відредагуйте назву (за замовчуванням — «Довідник ПК») і натисніть **«Додати»**.

На робочому столі зʼявиться іконка з логотипом ДДАЕУ. Запуск з неї відкриває застосунок **на весь екран, без панелей браузера** — як звичайний додаток. Усе це вже налаштовано в `index.html` (теги `apple-touch-icon`, `apple-mobile-web-app-capable`, маніфест).

> На Android (Chrome) аналогічно: меню «⋮» -> «Встановити застосунок» / «Додати на головний екран».

## Прев'ю при пересиланні посилання в месенджері
У файлі є мета-теги Open Graph і картинка `og-image.png` — посилання показуватиме картку (назва, опис, логотип). Для гарантованої роботи в усіх месенджерах після деплою бажано прописати **абсолютні URL**: замініть у `index.html` `content="og-image.png"` на повний шлях, напр. `content="https://www.dsau.dp.ua/pk2026/og-image.png"` (для `og:image` і `twitter:image`), і додайте `<meta property="og:url" content="https://www.dsau.dp.ua/pk2026/">`. Якщо прев'ю «залипло» — оновіть кеш через developers.facebook.com/tools/debug (Scrape Again).

## Експорт у PDF / друк
Угорі праворуч — кнопка **«Друк / PDF»**: розгортає всі розділи в один документ, далі «Зберегти як PDF».
